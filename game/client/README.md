# OldSchoolCraft — Client

Minecraft 1.7.10 · Forge `10.13.4.1614` · Java 8

## Install
1. Create a MultiMC/Prism instance: MC 1.7.10 + Forge `10.13.4.1614`.
2. Download every jar in [`modlist.txt`](modlist.txt) into the instance `mods/`.
3. Copy `config/` and `scripts/` into the instance.

The 37 mods in [`../client-only-mods.txt`](../client-only-mods.txt) are the
client-only extras that are intentionally absent from the server build.
